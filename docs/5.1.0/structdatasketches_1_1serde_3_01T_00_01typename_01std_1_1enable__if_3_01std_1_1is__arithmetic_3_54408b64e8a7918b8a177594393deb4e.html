Detailed Description

template<typename T>
struct datasketches::serde< T, typename std::enable_if< std::is_arithmetic< T >::value >::type >

serde for all fixed-size arithmetic types (int and float of different sizes).

in particular, kll_sketch<int64_t> should produce sketches binary-compatible with LongsSketch and ItemsSketch<Long> with ArrayOfLongsSerDe in Java
